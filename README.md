# Task Manager (To-Do List)

A simple task manager built with JavaScript that allows users to add, complete, and delete tasks. Tasks are saved using localStorage.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persist data using localStorage

## 🔗 Live Demo

https://task-manager-js-pearl.vercel.app

## 🛠️ Technologies

- HTML
- CSS
- JavaScript

## 💡 What I learned

- DOM manipulation
- Event handling
- localStorage usage
- Structuring JavaScript code
